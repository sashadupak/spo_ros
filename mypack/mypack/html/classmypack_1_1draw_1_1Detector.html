Detailed Description

Класс ноды Detector.

Читает - /camera/color/image_raw - обычное изображение с камеры
Публикует в /object_detection/output - исходное изображение, с холстом обведенным рамкой
Публикует в /object_detection/coords - координаты пикселей-вершин
Параметры first, second - параметры фильтра Кэнни
Параметры thrsh_low, thrsh_high - параметры площади холста
